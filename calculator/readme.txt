Project Title: Responsive Calculator using HTML & CSS
Description:
This project is a simple and responsive calculator designed using only HTML and CSS. It provides a clean and user-friendly interface for performing basic arithmetic operations. The layout is fully responsive and works smoothly on desktop, tablet, and mobile devices.
Features:


Responsive design


Clean and modern UI


Button-based layout


Supports basic calculator structure


Styled using CSS Grid


Beginner-friendly project


Technologies Used:


HTML5


CSS3


How to Run:


Download or clone the repository


Open the project folder


Run the index.html file in any web browser


Folder Structure:


index.html → Calculator structure


style.css → Styling and responsiveness


README.txt → Project information


Purpose of Project:
This project was created for practice to improve frontend skills in HTML and CSS and to understand responsive layouts.
